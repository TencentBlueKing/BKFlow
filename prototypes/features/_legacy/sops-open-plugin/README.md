# sops-open-plugin

## Purpose

This folder archives the historical Standard Ops open-plugin prototype pages that used to live under `prototypes/output/`.

## Contents

- `index.html` is the archive landing page.
- `main-flow.html` preserves the main flow prototype.
- `space-open-plugin-management.html` preserves the space management prototype.
- `task-plugin-error-state.html` preserves the task error state prototype.
- `template-plugin-selection.html` preserves the template plugin selection prototype.
